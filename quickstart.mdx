This quickstart takes you from an empty workspace to a live app. You only need a
Genie account and an idea to describe.

## Prerequisites

Before you begin, you need:

- A Genie account (sign up at [genie-app.de](https://www.genie-app.de)).
- A modern browser — no local install required. Genie builds and deploys in the
  cloud.

## Get started

<Steps>
  <Step title="Finish onboarding">
    On first sign-in, a short guided flow collects your name, account type, and
    role, then turns a personalized welcome prompt into your first project and
    build. See [Workspace → Onboarding](/features/workspace#onboarding).
  </Step>
  <Step title="Describe your app">
    From the workspace dashboard, type what you want to build into the prompt
    input — for example, *"a project tracker with a kanban board and a stats
    dashboard."* Quick actions offer starting points if you're not sure where to
    begin.
  </Step>
  <Step title="Review the plan">
    Genie proposes a plan before writing any code. Read it, request changes, or
    approve it. Approving starts the build. See
    [Building apps → Planning](/features/building-apps).
  </Step>
  <Step title="Watch it build and deploy">
    The build pipeline shows live step progress. A scoped coding agent writes the
    code, Genie provisions a [Genie Cloud](/features/genie-cloud) backend
    (database, storage, auth) as needed, and deploys a live preview.
  </Step>
  <Step title="Iterate with Genie AI">
    Open [Genie AI chat](/features/genie-chat) and keep refining — ask for
    changes, new pages, or backend tweaks. Genie AI can read and edit everything
    in your workspace on your behalf.
  </Step>
</Steps>

## Connect over MCP

Genie exposes its own [MCP server](/features/mcp-and-tools) so external AI
clients (Claude Desktop, Cursor, custom agents) can manage your workspaces and
projects programmatically. Authorize a workspace over OAuth and point your
client at:

```
https://www.genie-app.de/api/mcp
```
