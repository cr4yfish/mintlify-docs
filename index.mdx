title: "Introduction"
description: "Genie is an AI app builder that plans, codes, and deploys full-stack web apps from natural-language prompts."
---

Genie turns a prompt into a running web app. You describe what you want in a
workspace, and Genie plans the work, writes the code, provisions a backend, and
deploys the result — then keeps iterating with you through **Genie AI**, the
built-in chat assistant.

This site documents Genie for the people and agents who build on it: the product
features users see, and the internal platform that powers them. It is the source
that Genie's own internal AI agents query over MCP, so it aims to be complete
across both surfaces.

## What Genie does

<Steps>
  <Step title="You prompt">
    Describe the app you want from the workspace dashboard or in Genie AI chat.
  </Step>
  <Step title="Genie plans">
    Genie proposes a plan you can review and approve before any code is written.
  </Step>
  <Step title="Genie builds and deploys">
    A scoped coding agent runs the build pipeline, provisions a Genie Cloud
    backend, and deploys a live preview.
  </Step>
  <Step title="You iterate">
    Keep refining through chat — Genie AI can read and edit everything in your
    workspace on your behalf.
  </Step>
</Steps>
